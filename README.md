# Job_Portal
A full-stack MERN Job Portal application where users can apply for jobs and companies can post job openings.

## Features
- User & Company Authentication (JWT)
- Job Search & Apply
- Company Job Posting
- Protected Routes using Middleware

## Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
